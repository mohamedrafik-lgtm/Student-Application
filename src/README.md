# Student App - Taiba Center

## تطبيق الطلاب - مركز طيبة

تطبيق إدارة الطلاب في مركز طيبة التعليمي مع نظام تسجيل دخول متكامل.

### الميزات المُنجزة ✅

1. **صفحة تسجيل الدخول**
   - تصميم عربي RTL
   - التحقق من صحة البيانات
   - دعم الرقم القومي (14 رقم)
   - رسائل خطأ واضحة

2. **التكامل مع API**
   - اتصال بـ endpoint: `/api/trainee-auth/login`
   - معالجة الاستجابات والأخطاء
   - عرض معلومات المتدرب عند النجاح

3. **مبادئ SOLID**
   - فصل الاهتمامات (Separation of Concerns)
   - مكونات قابلة لإعادة الاستخدام
   - خدمات منفصلة للـ API

### التكوين المطلوب

1. **عنوان الخادم**: قم بتحديث `BASE_URL` في `src/services/apiConfig.ts`
2. **اختبار الاتصال**: تأكد من تشغيل الخادم على العنوان المحدد

### الاستخدام

1. أدخل الرقم القومي (14 رقم)
2. أدخل كلمة المرور
3. اضغط "تسجيل الدخول"

### الملفات الهامة

- `src/screens/LoginScreen.tsx` - صفحة تسجيل الدخول
- `src/services/authService.ts` - خدمة المصادقة
- `src/types/auth.ts` - أنواع البيانات
- `src/components/` - المكونات القابلة لإعادة الاستخدام
